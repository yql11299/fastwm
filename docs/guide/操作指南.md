## 1. 登录与认证

### 1.1 访问系统

打开浏览器访问 `http://服务器IP:5173`，首次访问会自动创建用户。

### 1.2 用户认证

系统采用基于 Cookie 的轻量认证机制：
- 首次访问时自动创建用户
- 用户信息存储在 `server/data/users/{userId}/settings.json`

## 2. 首页 - 证件管理

首页展示常用证件列表，支持水印文字输入和批量处理。

### 2.1 证件列表显示

- 证件按**布局分组**显示，与布局调整页面一致
- 每行显示多个证件卡片
- 点击证件可**选中/取消选中**

### 2.2 选择证件

- **单选**：直接点击证件
- **多选**：按住 Ctrl/Cmd 点击可选择多个
- **全选**：点击"全选"按钮

### 2.3 添加证件到常用列表

1. 点击"**添加证件**"按钮
2. 在弹出的文件浏览器中选择目录和文件
3. 点击"**添加**"按钮确认

> **注意**：
> - 支持 JPG、PNG、PDF 格式
> - 可以直接拖拽本地文件到添加窗口

### 2.4 水印文字输入

在首页中央的水印输入框中输入水印文字：
- 支持中英文混合
- 实时预览效果

### 2.5 一键生成

1. 确保已选择要处理的证件
2. 输入水印文字
3. 点击"**一键生成**"按钮
4. 等待处理完成后自动下载

## 3. 布局调整

点击顶部导航的"**调整布局**"进入布局调整页面。

### 3.1 拖拽调整位置

- **拖拽证件卡片**可以调整其在同一行内的位置
- **跨行拖拽**：将证件拖到其他行
- **创建新行**：拖拽到"拖拽到此处创建新行"区域

### 3.2 删除证件

将证件拖拽到底部的"**垃圾桶**"区域即可从常用列表移除。

### 3.3 添加证件

点击"**添加证件**"按钮，操作方式与首页相同。

### 3.4 重命名功能

重命名功能允许修改证件的显示名称：

1. 点击"**重命名**"开关，开启编辑模式
2. 点击要重命名的证件，证件名称变为可编辑输入框
3. 输入新名称
4. 按 **Enter** 或**失焦**保存
5. 再次点击"重命名"开关关闭编辑模式

> **注意**：
> - 空字符串不能作为名称
> - 检测到重名时会弹出确认提示
> - 修改后需要点击"**保存布局**"才会永久保存到服务器

### 3.5 保存布局

完成调整后，点击"**保存布局**"按钮将布局保存到服务器。

## 4. 画布水印预览

### 4.1 加载背景

1. 点击"**加载背景**"按钮
2. 选择服务器上的 PDF 或图片文件作为背景
3. 背景会显示在画布上

### 4.2 水印参数调整

在右侧属性面板中调整水印参数：

| 参数 | 说明 | 范围 |
|------|------|------|
| X 坐标 | 水印中心点水平位置 | 0-1 |
| Y 坐标 | 水印中心点垂直位置 | 0-1 |
| 缩放 | 水印字体大小（相对于背景宽度） | 0-1 |
| 旋转 | 水印旋转角度 | 0-360° |
| 透明度 | 水印可见度 | 0-1 |
| 字体 | 水印使用的字体 | - |
| 颜色 | 水印文字颜色 | - |

### 4.3 水印预览

水印实时渲染在画布上，可通过：
- **拖拽**移动水印位置
- **缩放手柄**调整大小
- **旋转手柄**调整角度

### 4.4 变换手柄


## 5. 水印方案管理

### 5.1 访问方案管理

点击顶部导航的"**方案管理**"进入方案列表页面。

### 5.2 创建新方案

1. 在方案管理页面点击"**保存当前配置**"
2. 输入方案名称
3. 选择是否设为**预设方案**（首页可见）
4. 点击"**保存**"

### 5.3 加载方案

在画布页面：
1. 点击属性面板中的"**加载方案**"
2. 从列表中选择一个方案
3. 水印参数会自动应用到当前画布

### 5.4 导入/导出方案

- **导出**：点击方案卡片的下载图标，导出为 JSON 文件
- **导入**：点击"**导入方案**"按钮，选择 JSON 文件

## 6. 批量处理与导出

### 6.1 批量处理流程

1. **首页选择证件** → 勾选要处理的证件
2. **输入水印文字** → 在首页中央输入框输入
3. **点击一键生成** → 等待处理完成
4. **自动下载** → 处理完成后自动下载 ZIP 文件

### 6.2 输出格式

- 每个证件单独生成一个 PDF 文件
- 所有文件打包为 ZIP 下载
- 文件名格式可在设置中配置

### 6.3 文件命名规则

| 规则 | 输出格式 | 示例 |
|------|----------|------|
| original | 原名 | `身份证.pdf` |
| timestamp | 原名_时间戳 | `身份证_20260428120000.pdf` |
| text | 原名_水印文字 | `身份证_仅供内部使用.pdf` |
| timestamp_text | 原名_时间戳_水印文字 | `身份证_20260428120000_仅供内部使用.pdf` |

## 常见问题

### Q: 为什么添加的证件在首页不显示？

A: 请检查：
1. 是否点击了"**保存布局**"
2. 文件格式是否为支持的 JPG、PNG、PDF

### Q: 水印预览与实际导出效果不一致？

A: 请确保：
1. 前端和后端都使用相同的水印参数
2. 背景文件路径正确

### Q: 如何修改文件命名规则？

A: 在首页点击用户设置，选择命名规则。

### Q: 忘记了水印文字怎么办？

A: 在"方案管理"页面，之前保存的方案会记录水印文字。
